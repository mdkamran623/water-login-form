# 🌊 Inside Water - Animated Login Form

A stunning, fully responsive animated login form with a water ripple effect. Built with HTML, CSS, and JavaScript.

![Preview](https://images.unsplash.com/photo-1555412654-72a95a495858?ixlib=rb-1.2.1&auto=format&fit=crop&w=634&q=80)

---

## ✨ Features

- 🌊 **Water Ripple Effect** - Interactive ripple animation on background
- 🎯 **Smooth Animations** - Scale-in and tracking animations
- 📱 **Fully Responsive** - Works on all devices (mobile, tablet, laptop, desktop)
- ✅ **Form Validation** - Email & password validation with user feedback
- 🎉 **Thank You Message** - Shows success message after login
- 🎨 **Modern UI** - Transparent glass-morphism design
- 🔄 **Interactive** - Click user icon to reveal login form
